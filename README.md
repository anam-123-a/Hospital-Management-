# 🏥 Hospital Analytics Dashboard

An interactive **Hospital Analytics Dashboard** built using **HTML, CSS, JavaScript, and Chart.js**.  
This project visualizes key healthcare data to provide meaningful insights for better decision-making.

---

## 🚀 Features

- 📊 KPI Cards (Admissions, Bed Occupancy, LOS, Revenue)
- 📈 Interactive Charts (Bar, Line, Doughnut)
- 🏥 Department-wise Filtering
- 📋 Performance Table with Status Indicators
- 💡 Key Insights & Recommendations Section
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3

---

## 📊 Dashboard Insights

- Tracks patient admissions trends (monthly)
- Monitors bed occupancy rate vs target
- Analyzes department-wise performance
- Highlights critical areas needing attention

---

## 📂 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Explore the dashboard 🎉
